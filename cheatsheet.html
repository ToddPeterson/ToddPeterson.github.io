Git Cheatsheet

Create

Create a new local repository.

git init

Clone an existing repository.

git clone [url]

Local Changes

List changed files.

git status

Diff tracked but not staged files.

git diff

Diff staged files.

git diff --staged

Add file (in its current state) to staging area. Use '.' to add all changes from the current directory.

git add [file]

Unstage a file, retaining changes.

git reset [file]

Commit all local changes from staging area.

git commit -m "[message]"

Change the last commit.

git commit -amend

Branch and Merge

List branches.

git branch

Create a new branch.

git branch [name]

Switch to a branch.

git checkout [branch]

Delete a local branch.

git branch -d [branch]

Merge a branch with the currently selected one.

git merge [branch]

Publish

List all remotes.

git remote -v

Show information about a remote.

git remote show -v

Download changes, but don't integrate.

git fetch [remote]

Download and merge/integrate changes.

git pull [remote] [branch]

Publish local changes on a remote

git push [remote] [branch]
