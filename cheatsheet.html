Git Cheatsheet

Create

Create a new local repository.

git init

Clone an existing repository.

git clone [url]

Local Changes

List changed files.

git status

Diff tracked files.

git diff

Add file to staging area. Use '.' to add all changes from the current directory.

git add [file]

Commit all local changes from staging area.

git commit -m "[message]"

Change the last commit.

git commit -amend
